# Nodesole

Nodesole is a visual, node-base editor that allows you to create simple console applications. It's main purpose is to teach people programming and algorithmic thinking without the need to learn the syntax of complex programming languages.

## How to build

In order to run the project locally you need to have nodejs installed and run the following commands:
```bash
    # Now you need to install the dependencies
    npm install

    # Compile the project
    npm run dev
```
